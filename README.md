# Проект "Noob Chat"

---
## Этот проект представляет собой веб-мессенджер

---

## Функционал

+ регистрация
+ авторизация
+ изменение данных
+ добавить аватар
+ выйти из аккаунта
+ валидация всех форм
+ чат

## Технологии

- Vite
- TypeScript
- Express
- SCSS
- Handlebars
- Stylelint
- ESLint

## Запуск проекта

Сборка проекта

```bash
    npm run build 
```
Запуск версии для разработчика

```bash
    npm run dev 
```

Сборка и запуск статического сервера на Express:

```bash
    npm run start
```
